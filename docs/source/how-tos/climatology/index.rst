Climatology How-tos
===================

This section contains how-to guides for calculating climatological means from ERA5 data using
earthkit-transforms. Each guide is an interactive Jupyter notebook that demonstrates a specific
workflow for working with ERA5 data and calculating climatologies. The examples cover various
aspects of climatology calculations, including calculating daily and monthly means, masking
data with geometries, and calculating area means.
