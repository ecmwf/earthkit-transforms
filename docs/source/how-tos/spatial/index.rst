Spatial How-tos
===============

This section contains how-to guides for working with spatial data from ERA5 using earthkit-transforms.
Each guide is an interactive Jupyter notebook that demonstrates a specific workflow for working with
ERA5 data and performing spatial transformations. The examples cover various aspects of spatial data
processing, including masking data with geometries, calculating area means, and working with spatial
subsets of data.
